队列

队列,先进先出.排过队吗?按顺序添加和处理的任务,都可以用队列的结构进行存储和消费.

class Queue {
2  constructor () {
3    this._items = {}
4    this._count = 0
5    this._lowestCount = 0
6  }
7 
8  enqueue(e) {
9    this._items[this._count] = e
10    this._count++
11  }
12  dequeue() {
13    if(this.isEmpty()) return undefined
14    const r = this._items[this._lowestCount]
15    delete this._items[this._lowestCount]
16    this._lowestCount += 1
17    return r
18  }
19 
20  isEmpty() {
21    return this._lowestCount === this._count
22  }
23  peek() {
24    return this.isEmpty()  ? undefined : this._items[this._lowestCount]
25  }
26 
27  size() {
28    return this._count - this._lowestCount
29  }
30 
31  clear() {
32    this._items = {}
33    this._count = 0
34    this._lowestCount = 0
35  }
36  toString() {
37    if(this.isEmpty()) return ''
38    let r = ''
39    for (const iterator of Object.values(this._items)) {
40      r += r === '' ? `${iterator}` : `, ${iterator}`
41    }
42    return r
43  }
44}

普通队列简单,但是有些场景需要对最新入队的元素进行操作.例如,针对需要存储一系列操作的需求.此时,需要灵活处理队首和队尾的数据内容.

当引发撤销操作的时候,操作队列可以从尾部弹出最后的操作记录.

我们需要双端队列.

class Deque extends Queue{
2  constructor () {
3    super()
4  }
5  addFront(e) {
6    if(this.isEmpty()) {
7      this.enqueue(e)
8    } else if(this._lowestCount > 0) {
9      this._lowestCount--      
10      this._items[this._lowestCount] = e
11    } else {
12      for (let i = this._count;i>0;i--) {
13        // 往后移动一位
14        this._items[i] = this._items[i - 1]
15      }
16      this._count++
17      this._lowestCount = 0
18      this._items[0] = e
19    }
20  }
21	// 从队尾出队
22  removeBack() {
23    if(this.isEmpty()) return undefined
24    const lastOne = this._items[this._count - 1]
25    if(this.size() === 1) {
26      this.clear()
27    }
28    delete this._items[this._count - 1]
29    this._count--
30    return lastOne
31  }
32 
33  peekBack() {
34    return this._items[this._count]
35  }
36}

接下来是回文检查,什么是回文字?

回文是指正反序都相等的字符串序列,例如 lol,madam等等.

最简单的方式就是使用双端队列来处理这个问题.

function palindromeCheaker(str) {
2  if(str === undefined || str === '' || str === null) return false;
3  const deque = new Deque();
4  [...str].forEach(i => deque.enqueue(i));
5  while(deque.size() > 1) {
6    if(deque.removeBack() !== deque.dequeue()) return false
7  }
8  return true
9}
10 
11console.log(palindromeCheaker('121'), palindromeCheaker('madam'), palindromeCheaker('jay'))
12// output
13// true, true, false

链表

存储多个元素,数组可能是最常用的数据结构,如果需要从起点或者中间插入元素,数组的操作成本很高.尽管JavaScript数组支持了一些方法来做这些事,但是背后的情况同样如此.

数组的元素在内存中是连续的,链表则可以是不连续的,链表的关键是使用节点的属性保存下一个或者上一个链表的信息.

相比于传统数组,链表添加或者移除一个元素不需要移动其他元素,大大降低了内存成本.

来思考一个算法题目,翻转链表:

题意：反转一个单链表。

示例: 输入: 1->2->3->4->5->NULL 输出: 5->4->3->2->1->NULL

直接翻转指针,可以避免多余的链表创建和内存占用.

// data is a LinkedList
2function reverseLinkList(data) {
3  if (data.size() > 1) {
4    let current = data.head
5    let prev = undefined
6    let next = undefined
7    while (current !== undefined) {
8      next = current.next;
9      current.next = prev;
10      prev = current
11      current = next
12    }
13    data.head = prev
14    return data
15  } else {
16    return data
17  }

8class DoublyLinkedList extends LinkedList {

14    const node = new DoublyNode(element)

16      this.head = node
17      this.tail = node
18    } else {
19      let current = this.head
20      while (current.next) {
21        current = current.next
22      }
23      current.next = node
24      node.prev = current
25    }
26    this.tail = node
27    this.count++
28  }
29  insert(element, index) {
30    const node = new DoublyNode(element)
31    if (index >= 0 && index <= this.count) {
32      if (this.count === 0) {
33        this.head = node
34        this.tail = node
35      } else {
36        let oldNode = this.getElementByIndex(index)
37        console.log(oldNode.element, 'is old node');
38        // 新插入节点设置了前后节点
39        node.next = oldNode
40        node.prev = oldNode.prev
41        // 旧的节点设置了 prev
42        oldNode.prev = node
43        if (node.prev) {
44          // 如果前节点存在
45          node.prev.next = node
46        } else {
47          // 不存在则说明插入的是链表头
48          this.head = node
49        }
50      }
51      this.count++
52      return true
53    }
54    return false
55  }
56  getTail() {
57    return this.tail
58  }
59  /**
60   * 
61   * @param {number} index 1. 链表长为 1
62   * 2. 长不为 1 => 1.删除首个元素/ 2.删除最后元素 / 3.删除中间元素
63   */
64  removeAt(index) {
65    const node = this.getElementByIndex(index)
66    // 空链表和无效 index
67    if (this.count === 0 && node === undefined) return undefined
68 
69    // 链长 1,删除 1
70    if (this.count === 1 && node === this.head) {
71      this.head = undefined
72      this.tail = undefined
73      this.count = 0
74      return node
75    }
76    // 链长不为 1
77    // index 对应的 node 有效
78    if (node === this.head) {
79      this.head = node.next
80    } else if (node === this.tail) {
81      node.prev.next = undefined
82      this.tail = node.prev
83    } else {
84      node.prev.next = node.next
85      node.next.prev = node.prev
86    }
87    this.count--
88    return node
89  }
90}

双向链表增加了tail属性,保存了链表尾部元素,且对每个节点的结构,增加了prev属性保存前一个节点信息.

下面看看双向循环链表,其跟双向链表的区别在于,对首个元素的prev设置为最后一个元素.最后一个元素的next设置为首个元素.因此,需要稍微调整代码结构.

双向循环链表的关键在于处理新节点的prev和next值,只要不是空链表,则每一个节点都有这两个值.

接下来是有序链表.为了让节点之间保持顺序,我们可以修改insert方法,让插入的位置由内部计算得出.

class SortedLinkedList extends LinkedList {
2  constructor() {
3    super()
4  }
5  insert(element, index=0) {
6    if(this.isEmpty()) {
7      return super.insert(element, 0)
8    }
9    // 自定义方法定义插入位置,用默认 index 代替 index 的效果.
10    const pos = this.getIndexNextSortedElement(element);
11    return super.insert(element, pos)
12  }
13  
14  getIndexNextSortedElement(element) {
15    let current = this.head;
16    let i = 0;
17    // 遍历,直接判断大小,也可以重新定义一个比较函数
18    for(;i < this.size() && current; i++) {
19      if(current.element < element) {
20        return i
21      }
22      current = current.next
23    }
24    return i
25  }
26}

其他方法都是继承的,不需要改变.由于插入的位置程序内部通过特定的比较算法去判断,因此实现了链表的有序性.

在操作和查找一个有序的链表的场景之下,可以使用不同的查找算法提高查找效率.
